Project Description
Background
The New Challenge is a competitive platform at The New School that facilitates student innovation and learning by allowing them to submit their ideas and compete for funding. My task was to simply the project submission process, create an engaging landing page to encourage students to get involved, and to organize the information and events in a user friendly layout.

Execution
I conducted my own research by hosting interviews with students who had used the platform in the past. In these interviews, I obtained valuable information about students’ experiences – both the positive and the negative – in using The New Challenge website and platform. Based on this feedback, I employed a dynamic landing page to allow the web admin access to change the designs and messages that would display on the website. In similar fashion, I simplified the submission process into three simple steps at the bottom of the website, to create a seamless, hassle-free user experience. As a last addition to the website design, I recreated the website infrastructure so users could navigate through projects, events, and other information with ease.
